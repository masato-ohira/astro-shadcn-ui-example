# Astro Starter Kit: Basics

## 主要コマンド

| コマンド                         | 動作                             |
| -------------------------------- | -------------------------------- |
| npm run install                  | インストール                     |
| npm run dev                      | 開発開始                         |
| npm run ssg                      | 静的ビルド                       |
| M={コミットメッセージ} yarn push | ビルドと git-push を合わせて実行 |
